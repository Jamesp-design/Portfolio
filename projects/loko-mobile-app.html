Loko

The first end-to-end interior design experience.

  • UX Design
  • Mobile
  • AR

Brief

My final-year self-initiated project focused on identifying a real home-decorating problem and developing a full prototype solution.

Solution

Loko is a mobile app that uses AR to support inspiration, design, and purchase workflows in a clear, linear end-to-end journey.

Outcome

I built a full visual identity, interactive mobile prototypes, and technical experiments including real-time object detection and ARKit trials.
