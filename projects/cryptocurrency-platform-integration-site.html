Building a comprehensive system to help customers seemlessly integrate.

  • UX Design
  • Mobile App

Brief

The company enables banks to offer bitcoin to their customers. As part of their service, they provide an integration site and prototypes that demonstrate feature behavior and API calls.

Solution

I developed a full set of screens and prototypes to show how these features work in practice, helping communicate seamless integration paths for bank partners.

Overview

The output was a comprehensive set of detailed screens and prototypes backed by research and stakeholder support for a large user base.
